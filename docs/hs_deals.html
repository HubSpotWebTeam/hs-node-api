<async, static> batchUpdate(updates)

Update a group of deals

Parameters:
Name Type Description
updates array

Array of objects. objectId corresponds with a dealId. See Example below.
Returns:

If successful the promise will resolve with { updated: true }. Otherwise the promise will resolve with an error message.

Type
Promise
Example
const hs = new HubspotClient(props);
const updates = [{
 "objectId": 93630457,
 "properties": [
   {
     "name": "dealname",

     "name": "dealname",

 ]},
 {

 ]
